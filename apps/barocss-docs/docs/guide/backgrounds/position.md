# Background position

Utilities for background-position.

## Classes

| Utility | Description |
|--------|-------------|
| `bg-center` | center |
| `bg-top` | top |
| `bg-bottom` | bottom |
| `bg-left` | left |
| `bg-right` | right |

## Examples

```html
<div class="bg-[url('/img.jpg')] bg-center h-40"></div>
<div class="bg-[url('/img.jpg')] bg-top h-40"></div>
```
