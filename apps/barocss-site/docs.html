Getting Started

BaroCSS is a real-time CSS library that brings the power of Tailwind CSS's JIT mode to your development workflow without any build process.

Installation

npm install barocss

Basic Usage

import { decl } from 'barocss';
import { BrowserRuntime } from 'barocss/runtime/browser';

// Initialize runtime
const runtime = new BrowserRuntime();
runtime.init();

// Generate CSS
const css = decl('bg-blue-500 text-white p-4 rounded-lg');
console.log(css);

Core Concepts

Utility Classes

BaroCSS supports all standard Tailwind CSS utility classes including spacing, colors, typography, layout, and more.

Variants

Use variants like hover:, focus:, dark: to create responsive and interactive designs.

Custom CSS

Extend BaroCSS with custom utilities and plugins for your specific needs.

API Reference

decl()

The main function for generating CSS from utility classes.

BrowserRuntime

Browser runtime for injecting and managing CSS styles.

ServerRuntime

Server-side runtime for generating CSS during build time.
